O que é Git Bash? Git Bsh è um aplicativo para ambientes Microsoft Windows que fornece uma camada de emulação para uma ezperiência de linha de comando Git.Bash é um acrônico para Bourne Again Shell. Um shell é um aplicativo de terminal usado para interagir com um sistema operaciona por meio de comandos escritos.
O Visual Studio Code é um editor de código simplificado com suporte para operações de desenvolvimento como depuração, execução de tarefas e controle de versão.  Ele tem como objetivo fornecer apenas as ferramentas de que um desenvolvedor precisa para um ciclo rápido de depuração de construção de código e deixa fluxos de trabalho mais complexos para IDEs com recursos mais completos, como o IDE do Visual Studio.
